# 🚀 LogikaInformatikaAPI - .NET 10

Proyek ini adalah implementasi materi perkuliahan Logika Informatika ke dalam bentuk REST API profesional.

## 🛠️ Fitur & Modul
- **Modul 00:** Pemrosesan Tipe Data Dasar.
- **Modul 01:** Logika Pengajuan Kredit Bank.
- **Modul 02 & 05:** Generator Tabel Kebenaran Otomatis.
- **Modul 03:** Pembuktian Hukum De Morgan.
- **Modul 04:** Analisis Implikasi & Biimplikasi.
- **Modul 06:** Verifikasi Hukum Logika Fundamental.
- **Modul 07:** Identifikasi Tautologi & Kontradiksi.

## 💻 Teknologi
- **Bahasa:** C#
- **Framework:** .NET 10 (ASP.NET Core Web API)
- **Documentation:** Swagger UI
